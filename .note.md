# JOSUI 面試作品 — 改善清單

> 依優先度整理，完成後可逐步刪除或勾選。

## 第一優先：明顯 Bug

- **HEX 色盤 crosshair 位置**：`Color-picker.tsx` 中 crosshair 的 `left` / `top` 使用 `color.coords`（OKLCH 的 L/C/H）當成 HSL 的 Hue/Saturation 用，座標系不一致，需先轉成 HSL 再計算位置。
- **RGBA / HSLA Slider 數值**：當內部 `Color` 為 OKLCH 時，`coords[0/1/2]` 不是 sRGB 的 R/G/B 或 HSL 的 H/S/L；Slider 的 `value` 與 `handleRgbaChange` / `handleHslaChange` 應基於轉換後的空間，避免滑桿與實際顏色脫鉤。
